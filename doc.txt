Installation de Creo v4.0 étudiant

Installation sur Windows :
	Télechargez le fichier CreoSetup.zip	
	Si vous avez déja installé Creo : 
		Appuyez sur la touche windows de votre clavier, maintenez et appuyez sur la touche R
		Une boite de dialogue va apparaitre et rentrez y C:\ProgramData\PTC\Licensing 
		Si une erreur apparait disant que ce chemin n'est pas disponible, c'est que Créo ne s'est pas déjà installé et que vous pouvez donc passer à l'étape suivante.
		Si l'explorateur de fichier se lance, supprimez tout ce qui apparait dans la fenetre de l'explorateur de fichier
